# Getting Started
1. Install and run the [voicevox engine](https://github.com/VOICEVOX/voicevox_engine)
2. `git clone https://github.com/kcoopermiller/maid-os.git`
3. `cd maid-os && nix develop`
4. `cava -p cava/config`
5. `python main.py`
